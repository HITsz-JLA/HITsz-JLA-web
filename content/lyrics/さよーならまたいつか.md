1. **気のない**:指“无兴趣、敷衍、无生气、无精打采、心不在焉”，“気”是日语中的一个核心概念，其本身有表达“心情、情绪、意志、意识、精神”等，能够构成的表达有：

   | 词汇               | 注音                   | 含义               |
   | ------------------ | ---------------------- | ------------------ |
   | 気がない／気のない | きがない／きのない     | 无精打采、心不在焉 |
   | 気を付ける         | きをつける             | 注意、小心         |
   | 気付く／気が付く   | きづく／きがつく       | 注意到、察觉到     |
   | 気にする           | きにする               | 介意               |
   | 気になる           | きになる               | 在意、惦记、感兴趣 |
   | 気がある           | きがある               | 对某人有意思、喜欢 |
   | 気に入る           | きにいる               | 喜欢、满意         |
   | 気に掛ける         | きにかける             | 挂念、放在心上     |

4. **<ruby>砕<rt>くだ</ruby>く・<ruby>砕<rt>くだ</ruby>ける**：破碎，砕く是他动词，表”击碎“，砕ける是自动词，表示因外力冲击而破碎。

5. **<ruby>やがて<rt></ruby>**：副词，表示不久，常用的表”不久“的词汇还有**<ruby>直<rt>す</ruby>ぐ、<ruby>暫<rt>しばら</ruby>く、<ruby>間<rt>ま</ruby>も<ruby>無<rt>な</ruby>く**。但这些词汇各有侧重，**すぐ**表示”立刻、马上“，时间间隔极短；**やがて**表示”不久、最终、迟早“，表示从现在起经过一段时间，必然会到达某个状态；**しばらく**则表示”暂时、一会儿、一段时间“；**まもなく**表示”即将、不久“，比すぐ稍缓，多用于电车广播等正式用语。以上四个词汇的词义区分在某年N3考试中有记载，读者可自行思考答案：
   **‣**　再利用できるものをリサイクルしなければ、資源は(　 )枯れてしまいます。 
   **A**、すぐ **B**、やがて C、しばらく **D**、まもなく

6. **はたと**：突然、一下子、猛地

7. **<ruby>気儘<rt>きまま</ruby>・<ruby>我儘<rt>わがまま</ruby>**：随心所欲、任性

8. **そこかしこで<ruby>袖触<rt>そでふ</rt></ruby>れる**：“そこ”表示那里，”かしこ”表示处处，二者连用表示“各个地方、到处”，“袖触れる”即为其字面意思“衣袖互相触碰”的引申“拥挤”。

9. **したり<ruby>顔<rt>がお</ruby>で**：词源来自古语「したり（＝してやったり）」 “哈，我成功了，我赢了”的那种表情。现代常用于讽刺别人“自以为聪明”“洋洋得意”。

10. **<ruby>背中<rt>せなか</ruby>を<ruby>殴<rt>なぐ</ruby>り<ruby>付<rt>つ</ruby>けて**：「背中」表示背后，「殴り付ける」表示用力殴打（付ける用法见上期每周一曲），此处引申为"中伤"的含义。

11. **<ruby>的外<rt>まとはず</ruby>れ**：常用的评价语，表示离题，其中“的”指的是“箭靶”。

12. **<ruby>天国<rt>てんごく</ruby>・<ruby>地獄<rt>じごく</ruby>**：天堂・地狱，日语比喻常用词。

13. **<ruby>縄<rt>なわ</ruby>**：绳子，可以以地名「**<ruby>沖縄<rt>おきなわ</ruby>**」来记忆读音。

14. **<ruby>握<rt>にぎ</ruby>り<ruby>締<rt>し</ruby>める**：紧握，其中**「締める」**常用于组成连用形动词，表示“紧紧”，例如常见的**「<ruby>抱<rt>だ</ruby>き締める」**紧紧抱住、**「<ruby>掴<rt>つか</ruby>み締める」**紧紧抓住。此外，「締め切り」也是一个常用词，表示“期限、截止日期”

15. **<ruby>貫<rt>つらぬ</rt></ruby>け<ruby>狙<rt>ねら</rt></ruby>い<ruby>定<rt>さだ</rt></ruby>め <ruby>蓋<rt>けだ</rt></ruby>し<ruby>虎<rt>とら</rt></ruby>へ どこまでも行け**  ：这句在整首歌词中属于意象跳跃强、文学性高、语法非典型（诗歌结构） 的一句，其中，「貫け」表示“贯穿、贯彻到底”，「狙い」是常用词，表示“目标”，「狙い定め」即表示“锁定目标”，因此此分句译为“锁定目标并贯彻到底”，「蓋し」多用于古语，表示诚然、想必、大概，在这里使用增添古典、庄严的感觉；「虎へ」向着虎，“虎”在文学中通常表示巨大的危险、恐惧、逆境、挑战、命运与试炼的象征等。因此此分句可译为“去直面命运的试炼”。此句多用简体形「え」列命令形/可能形发音，简介且富有突进、爆发的节奏感，同时又结合古典与现代词法，富有日本传统文学气息。本句是本首歌歌词的核心意象，表现了直面危险、困难的勇气与决心。
